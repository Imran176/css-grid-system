Grid Notes

  • Grid Template:

    Defines the number of columns and rows in a grid

    grid-template-columns: number of columns in the grid grid-template-rows: number of rows in the grid

  • Grid Size:

    Defines the size of columns and rows in a grid

    grid-auto-column's: minmax(150px, auto); => set the grid column's min size to 150px and max size to auto grid-auto-rows: minmax(150px, auto); => set the grid row's min size to 150px and max size to auto

  • Grid's Single Item's Size:

    Defines the size for the selected grid item

    grid-column-start: 1; => start from vertical line 1 in the grid grid-column-end: 3; => end at vertical line 3 in the grid grid-row-start: 1; => start from horizontal line 1 in the grid grid-row-end: 5; => end at horizontal line 5 in the grid grid-column: 1 / 3; === grid-row-start: 1; grid-column-end: 3; grid-row: 1 / 5; = grid-row-start: 1; grid-row-end: 5;
